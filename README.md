# chart3

A small utility to assemble per-instrument sheet-music PDFs from individual song charts.

Given a set of PDFs (typically one per song, each containing multiple instruments),
the tool generates a single combined PDF containing only the pages relevant to a
specific instrument. This was built to simplify onboarding new band members by
giving them a complete, correct set of charts in one step.

Implementation is in PHP with a MySQL-backed catalogue.

Setup and usage details are documented in the wiki:
https://github.com/owen-kellie-smith/chart3/wiki
